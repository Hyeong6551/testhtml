태그 선택자

div { }, span{ } 등 등


전체 선택자

* { } 모두 적용


아이디 선택자

아이디 속성값은 중복이 안됨 (다중선언 불가)
id="aa" 면
#aa { }


클래스 선택자

클래스는 다중 선언 가능
class="bb"면
.bb { }


다중 클래스

class="aa bb"면
다중으로 .aa{ } .bb{ }


속성 선택자

대괄호 [ ]로 사용, 단독 사용 불가
사용예시 : 선택자[속성='값']


가상 클래스 선택자

사용예시 : 선택자:가상 클래스


가상 클래스 종류

hover : 커서 위에 올려 놓았을 때 이벤트 발생

active : 요소를 활성화시켰을 때 이벤트 발생 (ex 버튼 클릭)

checked : 요소가 체크되었을 때

disabled : 요소가 비활성 요소일 때

enabled : 요소가 활성 요소일 때

visited : 요소를 방문했을 때 스타일 반영
a 태그만 사용 가능

focus : 요소가 포커싱될 때 (tabindex)

nth-child(N) : 요소가 N번째 요소일 때 스타일 반영


결합자

선택자 1 ~ 선택자 2 (ex/ div ~ span)


결합자 종류

형제 결합자 : ~ 부호 사용
선택자 1 기준으로 모든 선택자 2에 적용

인접 형제 결합자 : + 부호 사용
선택자 1 기준으로 바로 뒤에 있는 선택자 2 에만 적용
중간에 다른 태그가 섞여있으면 적용안됨

자식 결합자 : > 부호 사용
선택자 1의 자식 요소

자손 결합자 : 띄어쓰기 사용
선택자 1의 자손 요소
자식을 포함한 모든 자손
