디스플레이

block 요소

display 속성이 block일 경우 블록요소라고 불림


inline 요소

요소의 폭과 높이가 요소의 내용과 동일
가로폭과 높이 설정 불가


inline-block 요소

요소의 폭과 높이가 요소의 내용과 동일
폭과 높이 설정 가능
font style에 영향


flexbox

display 속성이 flex일 경우
인라인 블록 요소를 통한 정렬을 대체하기 위해 탄생
강제적으로 정렬


justify-content

어떻게 정렬할 것인지 설정 (flex-start/end, center)
(space-between/around/evenly)


align-items

교차 축 설정, 기본값 stretch / (flex-start/end, center)


flex-direction 플렉스 박스 방향

column(세로), row(가로) 뒤에 reverse를 붙여 역방향 가능


gap 플렉스의 간격

간격 조정



grid

한번에 가로 세로를 동시에 조정


grid-template-columns 열 정의

fr : 남은 그리드의 공간 차지
repeat(횟수, 단위) -> 횟수를 auto-fill 가능 (반드시 고정폭 사용)


justify-content

기본 축 기준 아이템 위치 설정, 폭만 넓으면 됨


align-content

교차 축 기준 아이템 위치 설정, flexbox의 align-items와 헷갈리지 말 것
